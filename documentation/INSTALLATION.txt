 INSTALLATION STEPS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. EXTRACT THE ZIP FILE
   ├─ Locate the downloaded ZIP file: KaraokeStudioPro_v3_*.zip
   ├─ Right-click and select "Extract All..."
   └─ Choose extraction folder (e.g., C:\KaraokeStudioPro\)
      (You can place it anywhere on your system)

2. LOCATE THE EXECUTABLE
   ├─ After extraction, navigate to the folder
   ├─ Look for: KaraokeStudioProV3.exe (Windows application)
   └─ This is the main application file

3. RUN THE APPLICATION
   ├─ Double-click KaraokeStudioProV3.exe
   ├─ Application will launch (may take 5-10 seconds on first run)
   └─ Karaoke Studio Pro is ready to use!

4. CREATE A SHORTCUT (Optional)
   ├─ Right-click KaraokeStudioProV3.exe
   ├─ Select "Create shortcut"
   ├─ Move shortcut to Desktop or Start Menu
   └─ For quick access in future

 CONFIGURATION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Click "Settings" button to configure:
├─ FFmpeg Path          - Location of FFmpeg executable (usually auto-detected)
├─ yt-dlp Path          - Location of yt-dlp (usually auto-detected)
├─ Download Directory   - Where to save downloaded videos
├─ Measurement Mode    - Switch between dB Output (dBFS) or SPL Estimate
└─ Auto-Reduce Threshold - Set volume reduction trigger (0-100%, default 80%)

Most settings are auto-configured. Customize audio settings for your room!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 AUDIO CALIBRATION (v3 NEW FEATURE)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Karaoke Studio Pro v3 includes room-specific audio calibration:

MEASUREMENT MODES:
├─ dB Output (dBFS)      - Raw digital signal level (-80 to 0 dB)
│                          [Technical mode for debugging]
│
└─ SPL Estimate (Room)   - Room sound pressure (~60-90 dB SPL)
                          [Calibrated for typical karaoke rooms]

HOW TO CALIBRATE:
1. Open "Settings" → "🔊 Audio Settings"
2. Select measurement mode that works for your room
3. Set "Auto-Reduce Threshold" based on your preferred volume:
   ├─ Lower value (0-50%)   = More sensitive, reduces volume sooner
   ├─ Default (80%)         = Recommended starting point
   └─ Higher value (90-100%) = Less sensitive, allows louder levels

4. Test with actual karaoke sessions and fine-tune as needed
5. When threshold is reached, app automatically reduces volume by 5%

NOTE: Auto-reduce prevents excessive audio spikes and hearing damage!

❌ Application Won't Start
   ✓ Ensure you have Windows 7 SP1 or later
   ✓ Try running as Administrator: right-click KaraokeStudioProV3.exe → "Run as administrator"
   ✓ Check that all files extracted correctly
   ✓ Restart your computer and try again

❌ No Audio Output
   ✓ Check your system volume settings
   ✓ Verify audio device is connected and enabled
   ✓ Try a different video file to test
   ✓ Check audio levels indicator on right side

❌ YouTube Download Not Working
   ✓ Check your internet connection
   ✓ Verify YouTube URL is correct and video is available
   ✓ Try a different video
   ✓ Ensure download directory has enough disk space

❌ Video Playback Issues
   ✓ Check file format is supported (MP4, MKV, AVI, WebM)
   ✓ Try a different video file
   ✓ Ensure video file is not corrupted
   ✓ Check that file isn't in use by another application

❌ Application Hangs or Freezes
   ✓ Wait 10-20 seconds (may be loading)
   ✓ If still frozen, close with Task Manager (Ctrl+Shift+Esc)
   ✓ Restart the application
